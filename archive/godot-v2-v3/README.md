# Archived Godot/V2/V3 Tooling

This directory contains inactive Godot/V2/V3 client work and its historical QA
tooling. The active product client is the browser client, and root `qa:*`
package scripts no longer expose these archived parity runners.

Historical files moved from `bin/qa/` live in `bin-qa/`. They are retained for
reference only and may contain stale package-script names or paths from the
period when Godot/V2 parity work was active.

Some archived visual-comparison scripts used packages that are no longer active
root dependencies. Restore those dependencies deliberately if this workstream is
reactivated.
