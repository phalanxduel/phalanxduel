# Triage Labels

The following labels are used for issue management:

- `needs-triage`: Issue created, awaiting classification.
- `needs-info`: Awaiting clarification from the reporter.
- `ready-for-agent`: Fully specified, agent may pick this up.
- `ready-for-human`: Requires a human engineer's expertise.
- `wontfix`: Issue closed as out of scope or invalid.
