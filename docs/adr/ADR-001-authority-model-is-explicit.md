# DEC-2A-001: Authority model is explicit

Authority model is explicit: `docs/gameplay/rules.md` is normative rules authority; `shared/src/schema.ts` is contract authority; `engine/src/state-machine.ts` is runtime transition authority; `docs/architecture/principles.md` is descriptive and must match runtime/contracts.
